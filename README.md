# dialog-chat-app
## アプリの詳細
Twitter のようなコメントを登録し、タイムラインに表示する Web アプリケーション

## 技術
#### フロントエンド

|使用技術|詳細|
|-|-|
|Nuxt.js(SPA mode)|フロントエンドフレームワーク|
|vee-validate|フォームのバリデーション|
|Vuetify|UIコンポーネント|

#### バックエンド

|使用技術|詳細|
|-|-|
|Rails6(API)|APIサーバー|
|devise_token_auth/devise|ログイン/ログアウト、プロフィール・パスワードの編集で使用|
|PostgreSQL|データベース|
